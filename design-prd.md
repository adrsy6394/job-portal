# Design PRD

# Job Portal Management System

---

# 1. Design Goal

The goal of this design is to create a modern, premium, and professional recruitment platform with a clean visual hierarchy. The interface should feel minimal while highlighting important sections through strong typography, circular graphics, and a green-themed branding.

The overall design should closely follow the provided design reference.

---

# 2. Design Style

* Modern
* Clean
* Professional
* Minimal
* Corporate
* Spacious Layout

---

# 3. Design Theme

The website should use a recruitment-focused visual style.

Characteristics:

* Large bold headings
* Soft background colors
* Dark green branding
* Bright green accent color
* Rounded elements
* Circular decorative graphics
* Large whitespace
* Premium card-based sections

---

# 4. Color Palette

Dark Green

Used For:

* Navbar
* Headings
* Buttons
* Cards
* Icons
* Footer
* Highlights

---

## Accent Color

Used For:

* Primary Buttons
* CTA Buttons
* Active Elements
* Decorative Shapes

---

## Background Color

Used For:

* Entire Website Background
* Section Background

---

## Secondary Background

White

Used For:

* Cards
* Content Areas

---

## Text Colors

Primary Text

* Dark Green

Secondary Text

* Dark Gray

Muted Text

* Medium Gray

---

# 5. Typography

Heading Style

* Bold
* Uppercase where required
* Large font size
* Strong hierarchy

Sub Heading

* Medium Weight

Paragraph

* Clean
* Easy to read
* Comfortable line spacing

Button Text

* Medium Bold
* High Contrast

---

# 6. Layout Structure

Desktop Layout

* Wide Container
* Multi-column Grid
* Large White Spaces
* Symmetrical Alignment

Tablet Layout

* Two-column layout where possible

Mobile Layout

* Single-column layout
* Stacked sections
* Responsive spacing

# 9. Brand Logo Section

Display partner/company logos in a horizontal layout.

Requirements

* Equal spacing
* Monochrome logos
* Center aligned
* Clean appearance

* Large Heading

Below this

Two Information Cards

Card 2

* Large Padding
* Simple Icons/Illustrations

# 11. Category Section

Display job categories using circular cards.

Categories include:

* Legal
* Marketing
* Sales
* Human Resource
* Logistics
* Administration
* Product Development
* Technical
* Tech & IT (Center Highlight)

* Thin Circular Borders
* Equal Circle Sizes
* Center circle filled with Dark Green
* Other circles outlined
* Symmetrical Layout

# 14. Buttons

Primary Button

* Bright Lime Green
* Dark Text
* Rounded Corners
* Medium Padding

Secondary Button

* Dark Green
* White Text

Hover State

* Smooth Transition

---

# 15. Cards

All cards should have:

* Rounded Corners
* Soft Background
* Consistent Padding
* Equal Spacing
* Clean Borders

---

# 16. Decorative Elements

Use large decorative circles throughout the page.

Include:

* Large Background Circles
* Semi-transparent Shapes
* Layered Circular Graphics

These elements should match the reference design and should not distract from the content.

---

# 17. Spacing

Maintain generous spacing throughout the website.

Use consistent spacing between:

* Sections
* Cards
* Headings
* Paragraphs
* Buttons

The design should never feel crowded.

---

# 18. Border Radius

Buttons

* Medium Rounded

Cards

* Medium Rounded

Circular Elements

* Fully Rounded

---

# 19. Icons & Images

Images

* High Quality
* Professional
* Recruitment-related

Icons

* Simple
* Outline Style
* Minimal

---

# 20. Responsive Design

Desktop

* Multi-column Layout

Tablet

* Two-column Layout

Mobile

* Single-column Layout
* Responsive Typography
* Responsive Images
* Responsive Buttons
* Responsive Cards
* No Horizontal Scroll

---

# 21. Animation

Use subtle animations only.

Recommended:

* Fade In
* Smooth Hover
* Scale on Hover
* Smooth Button Transition
* Smooth Card Hover

Animations should be elegant and not distracting.

---

# 22. Design Consistency

Maintain consistency across the entire application.

Ensure:

* Same color palette
* Same typography
* Same spacing
* Same button style
* Same card style
* Same border radius
* Same icon style
* Same visual hierarchy

The final UI should closely match the provided reference image while maintaining a professional recruitment platform appearance.
