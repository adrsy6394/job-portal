# Product Requirements Document (PRD)

# Job Portal Management System

---

# 1. Project Overview

The Job Portal Management System is a web application that connects Job Seekers and Recruiters on a single platform. Job Seekers can create their profiles, search for jobs, and apply for available positions. Recruiters can create company profiles, post job openings, and review applicants. The Admin manages users, jobs, and job applications.

---

# 2. Project Goal

Build a responsive Job Portal that allows:

- Job Seekers to find and apply for jobs.
- Recruiters to post jobs and manage applicants.
- Admin to manage the entire platform.

# 4. Functional Requirements

## Authentication

### User Registration

Both Job Seekers and Recruiters can register using the application.

### User Login

Registered users can log in according to their role.

### Admin Login

Admin can log in to access the admin dashboard.

### Logout

Users can securely log out from the application.

---

## Job Seeker Features

### Profile Management

Job Seekers can:

- Add Name
- Add Education
- Add Skills
- Upload Resume

---

### View Jobs

Job Seekers can browse all available job listings.

---

### Search Jobs

Users can search jobs using:

- Job Title
- Location
- Category

---

### Apply for Jobs

Job Seekers can apply to available job postings.

---

### View Applied Jobs

Users can see the list of jobs they have applied for.

---

## Recruiter Features

### Recruiter Profile

Recruiters can create and manage their company profile.

Company profile includes:

- Company Name
- Company Details

---

### Post Job

Recruiters can create and publish new job postings.

---

### View Applicants

Recruiters can view applicants who have applied for their posted jobs.

---

## Admin Features

### Manage Users

Admin can manage:

- Job Seekers
- Recruiters

---

### Manage Jobs

Admin can manage all job postings.

---

### Manage Applications

Admin can manage all submitted job applications.

# 8. Scope

The project includes only the following features:

- User Registration
- User Login
- Admin Login
- Job Seeker Profile
- Recruiter Profile
- Resume Upload
- Job Posting
- View Jobs
- Search Jobs
- Apply for Jobs
- View Applied Jobs
- Recruiter Applicant View
- Admin Management
- Logout
- Responsive Design

---

# 9. Out of Scope

The following features are not included in this version:

- Email Notifications
- Password Reset
- Chat System
- Payment Gateway
- Job Recommendations
- Social Login
- Interview Scheduling
- Company Reviews
- Bookmark Jobs
- Analytics Dashboard

---

# 10. Success Criteria

The system is considered successful when:

- Users can register and log in successfully.
- Job Seekers can complete their profile and upload resumes.
- Recruiters can post jobs successfully.
- Job Seekers can search and apply for jobs.
- Recruiters can view applicants.
- Admin can manage users, jobs, and applications.
- The application works correctly on desktop, tablet, and mobile devices.
